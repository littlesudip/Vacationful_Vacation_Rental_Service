Vacationful Is a Online Based Vacation Rental System.


To run the project:

1. Run migration
```bash
python manage.py makemigrations
python manage.py migrate
```
2. Create super user
```bash
python manage.py createsuperuser
```
3. Run the server
```bash
python manage.py runserver
```
